# StrawberryPickingSolution
## 项目名称
基于Nebula的计算机智能辅助草莓采摘农业旅游解决方案

## 项目简介
近年来，草莓采摘旅游进入到了人们的视野。通过设计一款基于人工智能的软件实现草莓成熟度检测、农作物知识普及与数字化农业管理，确保游客尽快采摘到成熟度最高的草莓，普及中小学生农村经验教育，并能提高游客游玩体验，节省农场主人力雇佣成本。
